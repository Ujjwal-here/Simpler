## How to Run This Project

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the development server:**
   ```bash
   npm run dev
   ```
3. Open your browser and go to the URL shown in the terminal (usually [http://localhost:5173](http://localhost:5173)).
